# Architecture Recommendation Agent

Internal AI-powered tool to:
- Gather project requirements
- Recommend AWS architectures
- Visualize system design
- Generate Terraform starter code
- Export infrastructure as downloadable assets

## Features
- AI / fallback architecture generation
- Mermaid diagram rendering
- Terraform generation (provider, main, variables, outputs)
- Downloadable infrastructure package

## Tech Stack
- Next.js
- TypeScript
- TailwindCSS
- Anthropic API (optional)
